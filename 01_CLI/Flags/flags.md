# Flags Cheat Sheet
- Flags modify behavior: short `-l`/`-a`, long `--all`; combine short ones `ls -la`.
- `ls -l` long listing; `ls -l -a` shows hidden; order usually irrelevant for flags.
- Use double dash for long names: `ls --all` (not `-all`).
- Flags can take params: `ls --ignore snap`, `ls --ignore=ubuntu /home`; shorthand `-I snap` (no equals).
- `--help` on many commands prints usage.
